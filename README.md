# Анализ метаданных COVID-19 CXR (PySpark)

Курсовой проект по дисциплине «Инфраструктура Big Data».  
Цель — построить аналитическую систему для мониторинга COVID-19 по метаданным рентгеновских снимков с использованием PySpark.
## Структура проекта
project/
├── analysis.ipynb          # основной ноутбук с кодом
├── README.md               # эта инструкция
├── requirements.txt        # зависимости (для локального запуска)
└── output/                 # результаты (Parquet, графики) 

## Запуск в Google Colab

1. Загрузить файл `analysis.ipynb` в Colab.
2. При необходимости установить зависимости (ячейка с `!pip install ...` уже есть внутри ноутбука).
3. Последовательно выполнить все ячейки сверху вниз.
4. В конце выполнения появятся:
   - очищенный DataFrame в памяти,
   - сохранённый датасет в формате Parquet в папке `output/`,
   - все графики будут показаны в ноутбуке.
